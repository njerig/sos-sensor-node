# sos-sensor-node

This repository's code is part of [S.O.S., a Shower/Stall Occupancy System](https://njerig.me/projects/sos).

## Electronics

### Parts List
- WeMos D1 Mini v2
- PIR sensor
- Reed sensor
- 5V micro USB plug
